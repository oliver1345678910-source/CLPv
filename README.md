# 💜 CLP Admin Panel

A modern, lightweight, and highly customizable administration system for Roblox, designed to provide a clean experience while remaining powerful and easy to configure.

## ✨ Features

* 👑 Powerful command system
* 🎨 Modern and user-friendly interface
* ⚡ Lightweight and optimized performance
* 🔧 Easy installation and configuration
* 🛡 Permission and administrator management
* 📋 Organized command categories
* 🔍 Player management utilities
* 🌐 API and HTTP support
* 💜 Clean and customizable design
* 📦 Modular structure for easy expansion
* 🚀 Fast startup and responsive controls
* 🖥 Multiple interface tabs and panels
* ⚙ Simple UserId-based ownership system
* 🔄 Built with future updates and scalability in mind

## 📦 Setup

1. Ungroup the contents into their intended services.
2. Open **CLP_Bootstrap**.
3. Replace the default Admin UserId with your own.
4. Enable **Allow HTTP Requests** and **Enable Studio Access to API Services** in Game Settings.
5. You're ready to go!

## 🎯 Designed For

CLP Admin Panel is suitable for:

* Roleplay games
* Hangout games
* Sandbox experiences
* Community games
* Private projects
* Large-scale Roblox experiences

## 🚀 Goal

CLP Admin Panel aims to provide a modern administration experience that combines simplicity, customization, and performance without sacrificing functionality.
